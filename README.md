# S-CORT : 위험운전 모니터링 시스템

실시간 운전자 상태 모니터링 및 위험 운전 감지 대시보드 시스템입니다.

🌐 **Live Demo**: [https://drive-alpha-weld.vercel.app/](https://drive-alpha-weld.vercel.app/)

## 📋 프로젝트 소개

S-CORT는 운전자의 실시간 상태를 모니터링하고 위험 운전 패턴을 감지하여 사전에 알림을 제공하는 시스템입니다. 네이버 지도 API를 활용하여 운전자들의 위치를 실시간으로 시각화하고, 위급/위험 상황을 즉시 파악할 수 있습니다.

## ✨ 주요 기능

- **실시간 차량 위치 모니터링**: 네이버 지도 API를 통한 실시간 위치 추적
- **위험 운전 감지**: 음주, 졸음, 과속, 난폭운전 등 위험 상황 자동 감지
- **대시보드**: 운전자 상태, 통계, 차트를 한눈에 확인
- **알림 관리**: 위험 상황 발생 시 즉시 알림 전송
- **데이터 분석**: 운전 패턴 분석 및 리포트 생성

## 🛠️ 기술 스택

- **Frontend**: React 18, TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **UI Components**: Radix UI, Lucide React
- **Maps**: 네이버 지도 API (Naver Maps API)
- **Charts**: Recharts
